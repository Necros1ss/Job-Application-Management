# Environment Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `DATABASE_URL` | PostgreSQL connection string. | `postgresql://user:pass@localhost:5432/db` |
| `JWT_SECRET` | Secret key for access tokens. | `your-secret` |
| `JWT_REFRESH_SECRET` | Secret key for refresh tokens. | `your-refresh-secret` |
| `PORT` | Backend HTTP port. | `5000` |
| `NODE_ENV` | Runtime mode. | `development` |
| `MAIL_HOST` | SMTP server hostname. | `smtp.gmail.com` |
| `MAIL_PORT` | SMTP server port. | `587` |
| `MAIL_USER` | SMTP username. | `user@example.com` |
| `MAIL_PASS` | SMTP password. | `password` |
| `MAIL_FROM` | Default sender address. | `noreply@example.com` |

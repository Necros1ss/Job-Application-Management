# **Job Application Tracker**

This web application is designed to help users keep track of their job applications. It allows users to add, manage, and organize their job applications efficiently. Built with React + Vite client app and
PostgreSQL

## **Project Structure**
- `frontend/`: React + Vite client app
- `backend/`: Express + PostgreSQL API

## **How to Run the Project**


## **Environment Setup**
- Copy `frontend/.env.example` to `frontend/.env`
- Copy `backend/.env.example` to `backend/.env`
- Update database connection and JWT secret in `backend/.env`
